# System scripts

## hlfd-install.ps1

Install useful tools for Windows.
- `bat`: replace `cat`.
- `dust`: replace `du`.
- `rg`: replace `grep`.
- `lsd`: replace `ls`.
- `7zip`.
- `vim`.
